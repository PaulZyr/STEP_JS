Домашнее задание #4

Task 1

Description: Написать функцию, которая принимает 2 числа и возвращает -1, если первое меньше, чем второе; 1 – если первое больше, чем второе; и 0 – если числа равны.

TO-DO: Input Number 1 and Number 2

Task 2

Description: Написать функцию, которая вычисляет факториал переданного ей числа.

TO-DO: Input Number 1

Task 3

Description: Написать функцию, которая принимает три отдельные цифры и превращает их в одно число. Например: цифры 1, 4, 9 превратятся в число 149.

TO-DO: Input Number 1, Number 2, Number 3

Task 4

Description: Написать функцию, которая принимает длину и ширину прямоугольника и вычисляет его площадь. Если в функцию передали 1 параметр, то она вычисляет площадь квадрата.

TO-DO: Input Number 1, Number 2 to count rectangle
Number 1 only to count square

Task 5

Description: Написать функцию, которая проверяет, является ли переданное ей число совершенным. Совершенное число – это число, равное сумме всех своих собственных делителей.

TO-DO: Input Number 1 to check

Task 6

Description: Написать функцию, которая принимает минимальное и максимальное значения для диапазона, и выводит только те числа из диапазона, которые являются совершенными.
Используйте написанную ранее функцию, чтобы узнавать, совершенное число или нет.

TO-DO: Input MIN to Number 1
and MAX to Number 2

Task 7

Description: Написать функцию, которая принимает время (часы, минуты, секунды) и выводит его на экран в формате «чч:мм:сс».
Если при вызове функции минуты и/или секунды не были переданы, то выводить их как 00.

TO-DO: Input Hours 1 Minutes 1 Seconds 1

Task 8

Description: Написать функцию, которая принимает часы, минуты и секунды и возвращает это время в секундах.

TO-DO: Input Hours 1 Minutes 1 Seconds 1

Task 9

Description: Написать функцию, которая принимает количество секунд, переводит их в часы, минуты и секунды и возвращает в виде строки «чч:мм:сс».

TO-DO: Input Hours Minutes Seconds

Task 10

Description: Написать функцию, которая считает разницу между датами.
Функция принимает 6 параметров, которые описывают 2 даты, и возвращает результат в виде строки «чч:мм:сс».
При выполнении задания используйте функции из предыдущих 2-х заданий: сначала обе даты переведите в секунды, узнайте разницу в секундах, а потом разницу переведите обратно в «чч:мм:сс».

TO-DO: Input Hours Minutes Seconds
