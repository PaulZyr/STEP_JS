Задание 4

Создать html-страницу для отображения/редактирования текста.
При открытии страницы текст отображается с помощью тега div. При нажатии Ctrl + E, вместо div появляется textarea с тем же текстом, который теперь можно редактировать. При нажатии Ctrl + S, вместо textarea появляет div с уже измененным текстом. Не забудьте выключить поведение по умолчанию для этих соче- таний клавиш.

Press Ctrl+E for edit text
Press Ctrl+S for save text
